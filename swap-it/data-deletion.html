This page explains how to request deletion of user data associated with the Swap It. mobile app developed by SheepBeansGames.

1. 如何提出刪除要求1. How to Request Deletion1. 削除リクエストの方法1. 삭제 요청 방법

  1. 請寄信至 sheepbeansgames@gmail.com。
  2. 信件主旨請填寫：「該換了吧資料刪除要求」。
  3. 請提供你用於登入 App 的 Google 或 Apple 帳號 Email；若你只要刪除意見回饋資料，請提供當時填寫的 Email 或大約送出日期。
  4. 請說明你要刪除的資料範圍，例如雲端備份、意見回饋、帳號關聯資料，或全部可刪除資料。
  1. Email sheepbeansgames@gmail.com.
  2. Use the subject line: "Swap It. Data Deletion Request".
  3. Include the Google or Apple account email you use to sign in to the app. If you only want feedback data deleted, include the email used in the feedback form or the approximate submission date.
  4. Tell us what you want deleted, such as cloud backups, feedback records, account-linked data, or all deletable data.
  1. sheepbeansgames@gmail.comまでメールしてください。
  2. 件名は「Swap It. Data Deletion Request」としてください。
  3. アプリへのサインインに使用しているGoogleまたはAppleアカウントのメールアドレスを記載してください。フィードバックデータのみの削除を希望する場合は、フィードバックフォームで使用したメールアドレス、または送信したおおよその日付を記載してください。
  4. クラウドバックアップ、フィードバック記録、アカウント関連データ、または削除可能なすべてのデータなど、削除したい範囲をお知らせください。
  1. sheepbeansgames@gmail.com으로 이메일을 보내 주세요.
  2. 메일 제목은 "Swap It. Data Deletion Request"로 작성해 주세요.
  3. 앱 로그인에 사용하는 Google 또는 Apple 계정 이메일을 포함해 주세요. 피드백 데이터만 삭제하려는 경우, 피드백 양식에 입력한 이메일 또는 대략적인 제출 날짜를 포함해 주세요.
  4. 클라우드 백업, 피드백 기록, 계정 연결 데이터 또는 삭제 가능한 모든 데이터 등 삭제하려는 데이터 범위를 알려주세요.

2. 會刪除的資料2. Data We Delete2. 削除するデータ2. 삭제되는 데이터

  • Firebase Authentication 中與 App 相關的帳號關聯資料。
  • Firebase Storage 中與該帳號相關的雲端備份，包括備份 JSON 與已同步的貼紙圖片。
  • Cloud Firestore 中可識別或可定位到你的意見回饋紀錄。
  • 與刪除要求相關、且不再需要保留的客服處理紀錄。
  • App-related account records in Firebase Authentication.
  • Cloud backups linked to the account in Firebase Storage, including backup JSON files and synced sticker images.
  • Feedback records in Cloud Firestore that can be identified as yours.
  • Support records related to the deletion request when they are no longer needed.
  • Firebase Authentication内のアプリ関連アカウント記録。
  • Firebase Storage内でそのアカウントに関連付けられたクラウドバックアップ。バックアップJSONファイルおよび同期されたステッカー画像を含みます。
  • Cloud Firestore内であなたのものと識別できるフィードバック記録。
  • 削除リクエストに関連し、保管の必要がなくなったサポート対応記録。
  • Firebase Authentication의 앱 관련 계정 연결 데이터.
  • Firebase Storage에서 해당 계정과 연결된 클라우드 백업. 백업 JSON 파일 및 동기화된 스티커 이미지를 포함합니다.
  • Cloud Firestore에서 사용자와 식별 또는 연결 가능한 피드백 기록.
  • 삭제 요청과 관련되어 더 이상 보관할 필요가 없는 고객 지원 처리 기록.

  • Local data is stored on your device. You can delete it in the app or remove the app to delete local device data.

  • 特定のユーザーを識別できない匿名または集計済みの分析データは、製品統計および品質改善のために保持される場合があります。
